# stm32-inmp441-audio-detector
基于STM32F407VET6和INMP441的实时噪音监测系统，支持A加权分贝测量和200-500Hz频段能量占比检测
# STM32 INMP441 实时音频检测系统

一个基于STM32F4和INMP441 MEMS麦克风的轻量级实时音频检测系统，开箱即用。

## ✨ 功能特性
- ✅ 200-500Hz频段能量占比实时检测
- ✅ 滑动平均滤波，数值稳定无闪烁
- ✅ 极低资源占用：FFT处理时间<1.5ms @ 168MHz

## 🛠️ 硬件清单
| 器件 | 型号 | 说明 |
|------|------|------|
| 主控 | STM32F407VET6 | 兼容所有STM32F4系列 |
| 麦克风 | INMP441 | I2S数字输出，底部进音 |
| 显示屏 | 1.69寸SPI LCD（ST7735） | 可选，用于显示数据 |
| 其他 | 杜邦线、面包板 | |

## 📌 引脚接线
| INMP441引脚 | STM32F4引脚 | 说明 |
|-------------|-------------|------|
| VDD | 3.3V | 必须3.3V，不能用5V |
| GND | GND | |
| SCK | PB13 | I2S2_SCK |
| WS | PB12 | I2S2_WS |
| SD | PB15 | I2S2_SD |
| L/R | GND | 左声道输出 |

## 🚀 快速上手
1. 克隆本仓库
2. 用Keil MDK打开`MDK-ARM/VOICE.uvprojx`
3. 编译并烧录到开发板
4. 按照接线图连接INMP441
5. 上电后等待1秒，LCD会显示当前分贝值和频段占比

## 🎯 校准指南
### 分贝校准
1. 打开手机分贝计（推荐使用「噪音计」APP）
2. 将手机和麦克风放在同一位置，距离10cm
3. 播放1kHz标准音频，调整到70dB
4. 修改`audio.h`中的`CALIBRATION_OFFSET`，直到显示值和手机一致

### 灵敏度调整
- 提高占比灵敏度：减小`RATIO_SMOOTH`（如0.3）
- 提高数值稳定性：增大`RATIO_SMOOTH`（如0.5）
- 修改检测频段：调整`TARGET_BAND_START`和`TARGET_BAND_END`

### 性能参数
| 参数 | 数值 |
|------|------|
| 采样率 | 4000Hz |
| FFT点数 | 1024 |
| 频率分辨率 | 3.90625Hz |
| 可分析最高频率 | 2000Hz |
| 分贝测量范围 | 35~120dB |
| 系统响应时间 | ~250ms |
